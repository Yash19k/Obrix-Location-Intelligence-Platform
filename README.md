# Obrix: Location Intelligence Platform
# 🌍 Geospatial AI Platform

An AI-powered geospatial analytics platform that helps businesses, urban planners, and government agencies identify the most suitable locations for infrastructure and commercial development using spatial intelligence, machine learning, and interactive mapping.

---

## 🚀 Overview

The platform analyzes multiple geospatial factors such as:

- 📍 Population Density
- 🛣️ Road Accessibility
- 🏢 Nearby Infrastructure
- 🏬 Competitor Analysis
- 🌱 Land Use
- 🌦️ Environmental Factors
- 🚗 Travel Accessibility

It generates a **Site Readiness Score (0–100)** with AI-driven insights and recommendations to support faster and more informed location decisions.

---

## ✨ Features

- Interactive map-based location selection
- AI-powered Site Readiness Score
- Multi-factor spatial analysis
- Competitor and accessibility analysis
- Location comparison
- Investment hotspot detection
- Interactive dashboards
- Automated PDF reports
- REST API for geospatial analytics

5. Calculate Site Readiness Score
6. Generate recommendations
7. Visualize results
8. Export reports
